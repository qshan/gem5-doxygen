77 /*
78  * This class is needed to allow system call retries to occur for blocking
79  * system calls in SE mode. A retry fault will be generated by the system call
80  * emulation code if blocking conditions arise; the fault is passed up the
81  * function call chain into the CPU model where it is handled by retrying the
82  * syscall instruction on a later tick.
83  */
